# Responsive AI Components

## Instructions

1. **Component structure**
   - Design reusable AI-driven components
   - Separate logic from presentation
   - Support dynamic data and states

2. **Responsiveness**
   - Mobile-first layouts
   - Adaptive sizing and spacing
   - Ensure accessibility across devices

3. **AI interaction**
   - Handle streaming or async AI responses
   - Show loading, thinking, and error states
   - Optimize re-rendering for AI outputs

4. **Performance**
   - Minimize unnecessary re-renders
   - Lazy-load heavy components
   - Optimize animations and transitions

## Best Practices
- Keep components stateless where possible
- Provide clear feedback during AI processing
- Use skeleton loaders or placeholders
- Ensure graceful degradation
- Test across screen sizes and devices

Use this skill when building scalable, AI-powered interfaces that must remain fast, responsive, and user-friendly.
